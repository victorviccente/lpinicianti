As tecnologias para você iniciar e entrar no mercado de TI!

HTML

É uma linguagem de marcação, onde marcamos os elementos para definir quais informações a página vai exibir.

CSS

É uma linguagem de folha de estilo composta por “camadas”, criado com o propósito de estilizar as páginas HTML.

JavaScript

É uma linguagem de programação que permite a você implementar elementos dinâmicos em páginas web.

Python

É uma linguagem utilizada, como ciência de dados, desenvolvimento web, automação de tarefas e desenvolvimento de aplicativos.

React

É uma biblioteca JavaScript com foco em criar interfaces de usuário de forma "componetizada".

TypeScript

É um superset do JavaScript que incluir recursos que não estão presentes na linguagem, além de torná-la estática.

styled-components

É uma biblioteca que utiliza o conceito de CSS-in-JS, ou seja, que nos permite escrever CSS dentro do JS.

Tailwind CSS

É um framework do CSS que nos fornece classes utilitárias com o propósito de estilizar as páginas HTML.
